# TechSuite Replica

TechSuite is a high-fidelity digital replication of the enterprise portal for TechSuite Systems, built using modern web development practices. It highlights professional design, interactive service timelines, and modular pages.

## 🚀 Features

- **High-Fidelity UI**: Premium web design layout matching corporate mockup standards, featuring clean aesthetics, light mode styling, and dynamic typography.
- **Client-Centric Hero Section**: Vibrant color gradients and clean typographic layout.
- **About Us & Values**: Detailed grid layouts presenting core company values (Innovative, Human-focused, Experts, Customer First).
- **Alternating Services Timeline**: A custom-drawn vertical timeline listing core technology service portfolios (ERP, Cloud hosting, Analytics, AI automation, Cybersecurity, Smart spaces).
- **Reusable Service Page Template**: Dynamic page layout rendering individual service pages with customized content modules and CTAs.
- **Interactive Contact Form**: High-fidelity contact form featuring validation, tooltips, and interactive inputs.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router, Turbopack)
- **Runtime & Language**: React 19, TypeScript
- **Styling**: Tailwind CSS v4, PostCSS
- **Components**: Lucide React (Icons), React CountUp (Animations), React Hook Form (Validation)

## 📦 Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Jaswanth-droid/TechSuite-replica.git
   cd TechSuite-replica
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build the application for static export:
   ```bash
   npm run build
   ```

The output will be exported as static HTML to the `/out` directory.
